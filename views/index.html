API Project : Timestamp Microservice

user stories:
  1. ) I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)
  2. ) If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. ) If it does not contain a date or Unix timestamp, it returns null for those properties.
  4. ) If i don't pass any parameters, it will return current date.

Example Usage :

https://timestamp-avi.glitch.me/api/timestamp/December%2015,%202015

https://timestamp-avi.glitch.me/api/timestamp/1450137600

https://timestamp-avi.glitch.me/api/timestamp/

Example Output :

{"unixtime":1450137600000,"naturaltime":"Tue, 15 Dec 2015 00:00:00 GMT"}
